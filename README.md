Crea una sección con tres columnas, cada una con:

Ícono lineal minimalista (hoja, gota, escudo, etc.).

Bloque centrado con mensaje claro.

Diseño minimalista

Colores tierra (verde salvia, beige, marrón claro, off-white)

Tipografía sans-serif

Mucho espacio en blanco

Formas suaves y amigables
